# 拜访助手 — 智能备忘录+拜访记录语音助手

## 应用概述

一款移动端 APP，核心功能为"智能备忘录+拜访记录语音助手"，适合销售、客户经理、商务人员使用。

## 技术栈

- **框架**: Flutter（跨平台，支持 Android/iOS）
- **数据存储**: SQLite（sqflite）
- **语音识别**: speech_to_text 插件
- **本地通知**: flutter_local_notifications
- **语音解析**: 正则+关键词匹配（初版）

## 核心功能使用说明

### 一、每日待办清单

**位置**：底部导航 → 待办（第一个标签）

功能：
- 默认展示"未完成"列表，按截止时间倒序排列
- 点击 **添加待办** 按钮（右下角）创建新事项
- 勾选 ✅ 完成事项，自动移入历史
- 右上角过滤器可切换显示全部/未完成
- 超过截止日期的未完成事项显示红色"已过期"标记

### 二、语音录入拜访记录（核心功能）

**入口**：底部导航任一切面 → 点击 **麦克风** 按钮

**操作流程**：
1. 点击麦克风按钮 → 弹出语音录入浮层
2. 点击 **开始录音** → 说出拜访信息
3. 系统自动转录并解析 → 弹出结构化预览
4. 确认/编辑字段 → 点击 **保存记录**

**语音解析字段**：
- **拜访时间**（如"今天下午3点" → 自动转为具体时间）
- **参与人员**（自动提取人名列表）
- **拜访对象**（单位 + 姓名）
- **谈话内容摘要**（提取关键句）
- **结果**（达成意向/需跟进/初步接触/未果/待补充）

**自然语言示例**：
> "今天下午3点拜访了金川矿业的张总，和李经理、王工一起去的。聊了IPv6网络改造方案，客户对方案比较满意，初步达成合作意向，需要下周出详细报价。"

→ 自动解析出：时间=今天15:00, 单位=金川矿业, 联系人=张总, 参与人员=[李经理,王工], 结果=达成意向

**手动补录**：解析置信度低时自动进入编辑模式，支持手动填写所有字段。

### 三、拜访记录与待办联动

每条拜访记录保存后会自动生成一条跟进待办：
- 标题："跟进{联系人}"/"跟进{单位}"
- 截止时间：拜访日期后第3个工作日（跳过周末）
- 描述：附带拜访记录摘要

### 四、记录归档与搜索

**位置**：底部导航 → 记录（第二个标签）

功能：
- 时间线归档，按时间倒序排列
- 支持**搜索**（按单位、人员、内容关键词）
- 支持**结果筛选**（达成意向/需跟进等）
- 支持**日期范围筛选**

### 五、数据导出

**位置**：底部导航 → 设置 → 导出备份

将所有数据（待办+拜访记录）导出为 JSON 文件，存储到设备文档目录，可用于迁移或归档。
